Under the hood, Gatsby is a collection of APIs.

For example:

  • Pages are created with createPages
  • Data ‚Äúnodes‚Äù are created using sourceNodes

For a full list of APIs, see gatsby.dev/api

Creating pages in Gatsby

exports.createPages = ({ actions }) => {
  actions.createPage({
    path: '/my-page',
    component: require.resolve('./src/templates/page.js'),
  });
};

Adding data nodes in Gatsby

exports.sourceNodes =
  ({ actions, createNodeId, createContentDigest }) => {
    const myData = { name: 'Jason', likes: 'snacks' };
    actions.createNode({
      ...myData,
      id: createNodeId('MyData-jason'),
      internal: {
        type: 'MyData',
        contentDigest: createContentDigest(myData),
      },
    });
  };

We use these APIs to:

  • Load data from third-party sources
  • Create pages from files
  • Convert data to another format
  • And so much more!

In a lot of cases, this code is reusable.

For example:

  • Converting Markdown to HTML
  • Adding TypeScript support
  • Loading posts from WordPress

Gatsby plugins are shareable collections of code.
